# Hana's Bride To Be Party

One-page static e-invite for Hana's bridal shower. It can be hosted on GitHub Pages.

## Edit the details

Open `script.js` and update the `eventDetails` object at the top:

- `partyDate`
- `weddingDate`
- `venue`
- `address`
- `mapUrl`
- `schedule`
- names and invitation text

## Host on GitHub Pages

1. Push these files to a GitHub repository.
2. Go to the repository settings.
3. Open **Pages**.
4. Set the source to the main branch and root folder.
5. Share the generated GitHub Pages link with guests.
